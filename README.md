# 作業環境測定（1日法）学習用シミュレーター

仮想データ等から幾何平均・幾何標準偏差・第1評価値・第2評価値を計算し、管理区分を示す学習用シミュレーターです。

🔗 **[シミュレーターを開く](https://[your-username].github.io/[your-repo-name]/)**

---

## 概要

作業環境測定の1日法に基づき、A測定データから以下を自動計算します。

- 幾何平均 M
- 幾何標準偏差 σ₁（補正前）・σ（日間変動 0.084 補正後）
- 第1評価値 EA₁（log EA₁ = log M + 1.645 × log σ）
- 第2評価値 EA₂（log EA₂ = log M + 1.151 × (log σ)²）
- 管理区分（第1〜第3）の判定

B測定データがある場合は、A測定とB測定の総合判定も行います。

## 対象

- 作業環境測定士 登録講習（での検算・演習）
- 労働衛生コンサルタント 試験対策
- 産業衛生専攻医 学習
- 労働衛生・産業保健分野における作業環境測定の自己学習

## 機能

| 機能 | 内容 |
|------|------|
| A測定入力 | 最大10点まで入力可能 |
| B測定入力 | 最大3点まで入力可能（最大値のみ評価） |
| 計算過程表示 | log x・偏差・偏差²を有効数字10桁で表示 |
| 数直線 | 管理濃度・評価値の位置を視覚的に表示 |
| 判定マトリクス | A×B測定の3×3総合判定表 |

## 使い方

1. **管理濃度**（mg/m³）を入力
2. **A測定データ**を2点以上入力（推奨：6〜10点）
3. 必要に応じて**B測定データ**を入力（トグルをON）
4. **「管理区分を判定する」**ボタンを押す
